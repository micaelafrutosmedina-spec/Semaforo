# 🔊 Proyecto 1.º Año – Robótica Estática

## Nombre del Proyecto

**Alerta de Ruido en Aulas**


---

# 📌 Problemática

En muchas aulas el exceso de ruido dificulta la concentración de los estudiantes y la explicación del docente. Esto afecta el aprendizaje, reduce la atención y genera un ambiente poco adecuado para el desarrollo de las clases.

---

# 🎯 Objetivo

Diseñar un sistema que detecte automáticamente cuando el nivel de ruido en un aula supera un límite establecido y emita una señal de alerta para ayudar a mantener un ambiente más tranquilo.

---

# 💡 Descripción del Proyecto

El proyecto consiste en construir un dispositivo basado en **Arduino UNO** que mida continuamente el nivel de sonido mediante un sensor.

Cuando el ruido supere un valor previamente configurado, el sistema activará un **buzzer** que emitirá una señal sonora como advertencia. Una vez que el nivel de ruido vuelva a ser normal, la alarma se apagará automáticamente.

Este sistema busca fomentar un ambiente de estudio más adecuado de forma simple, económica y automática.

---

# 🛠 Materiales

* Arduino UNO
* Módulo sensor de sonido
* Buzzer
* Resistencia de 220 Ω
* Protoboard
* Cables jumper
* Cable USB para alimentación y programación

---

# 🔄 Funcionamiento

1. El sensor de sonido mide continuamente el nivel de ruido.
2. Arduino recibe la información del sensor.
3. Si el ruido supera el umbral configurado:

   * Se activa el buzzer.
4. Cuando el ruido disminuye:

   * El buzzer se apaga automáticamente.
